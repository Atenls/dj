# 「降厄」详解

迟迟没有发攻略，一方面是确实没有时间，另一方面也是担心降厄一世而亡，版本末搞那么复杂是否有点过于浪费精力。

所以没有很详细地去写一些原理性的东西，但是挑战在即，还是想尽可能地发布一些已有的思路、结论。为大家的挑战进程添砖加瓦。

想着没有详细的介绍可能会造成一些困扰，但回忆了一下上赛季的攻略，似乎也并没有太多讲解的地方，所以还是先发一下。

---

## 「暗影千机」赛季变更

降厄的思路和此前其实是基本一致的，在没有薄瘴后，很难说有什么不同。

确实也有所变化，但和《贴吧大神》说的什么 「*赤蝎给虫魄会加强橙武*」 不相关，实际上橙武虫魄几乎完全没利用空间，变量是千丝的CD调整带来的。

在上个赛季，虫魄的限制非常死，不仅是虫魄本身的约束，还有千丝、百足CD的限制。

在不鸣的加入后，这一方面导致了所有技能的CD调整空间都相对宽松，也使得残香犯错成本降低了。另一方面，千丝缩减的 `2s` 更是带来了较高的循环调整空间。

## 橙武循环

<span class="haste-tag-left">加速</span><span class="haste-tag-right">21131</span> <span class="haste-tag-left">延迟</span><span class="haste-tag-right">~35ms</span>

这套循环比起赛季初白皮书公布的循环高不少。

主要是针对 20~45ms 的情况进行了优化（由青衫完成），理论上来说，只要延迟不高于 `52ms` 其实都是可以稳妥地打出来的。

这赛季因为不僵的参与和其他DOT的调整，实际上蝎心的质量是大不如前了，因此所有循环末尾都出现了**要等**的情况。

所以除非你延迟低于 20ms ，能考虑不损失时间的情况下多打一个蝎心，否则基本上是不会有什么区别的。
